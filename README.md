### BTSH

This is a website for Black Top Street Hockey League.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Contributing

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Create a pull request
